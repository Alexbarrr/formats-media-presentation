# formats.media — Presentation

HTML-презентация, автоматически публикуемая через GitHub Pages.

**Живая версия:** публикуется автоматически при каждом push в ветку `main`.

## Как обновить презентацию

1. Отредактируйте `index.html`
2. Сделайте `git push` в ветку `main`
3. GitHub Actions автоматически задеплоит изменения (~1-2 минуты)
